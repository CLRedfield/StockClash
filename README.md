# 操盘杀 · StockClash

> 一局四分钟，跑完现实里四个月的行情。看谁是股神，谁在被割韭菜。

实时操盘竞速游戏：随机抽一只**真实历史标的**、从随机时刻开局，所有人同样起始资金，
逐秒跳动的真实 K 线里**做多做空 / 贷款加杠杆**，时间到结算，比谁钱最多。

## 特性

- 🕐 **时间压缩**：默认一局 4 分钟 ≈ 现实 4 个月，每秒刷新（档位/时长/窗口可调）。
- 📈 **真实行情**：打包真实历史日 K + 逐秒插值噪声，涨跌有"道理"（数据缺失自动回退合成走势）。
- 🎯 **经典 / 自定义模式**：建房时二选一卡片——经典一键套用推荐规则（1 股×100、休市、约 3 个月、道具），自定义自由调参。
- 🔊 **音效**：买卖 / 爆仓 / 道具 / 暴涨暴跌 / 胜负的合成音效，可一键静音（记忆偏好）。
- 📱 **移动端适配**：窄屏下大盘、资产 HUD 与买卖 / 快捷操作自适应排布。
- 🌐 **本地 / 联机**：本地对 AI，或联机开黑——联机沿用 hearthStoneKill2 的**公共 MQTT + 房主权威**，免服务器、免注册。
- 👑 **房主管理**：踢人、换座、增减 AI、逐座调 AI 难度。
- 💹 **多市场多选**：大 A / 港股 / 美股 / 虚拟货币，可多选，随机抽。
- 💸 **完整经济**：手续费、做多做空、保证金爆仓、贷款（放款即计 + 每 30 秒复利、欠债禁贷、结算扣本息）。
- 🎁 **道具模式（可选）**：每隔一段时间发小道具搅局（间隔可调），详见 [`道具图鉴.md`](道具图鉴.md)。
- 🎨 **StockClash 设计系统**：亮色交易终端风、金价金（`#c8932a`）强调、**绿涨红跌**、cool slate 中性色、hairline 卡片 + 软抬升阴影；Space Grotesk + Noto Sans SC + JetBrains Mono 字体，Lucide 线性图标（无 emoji）。设计 token 在 `src/styles/tokens/`。

## 运行

```bash
npm install
npm run dev      # 本地开发 http://localhost:5174
npm run build    # 生产构建到 dist/
```

## 技术栈

Vite + 原生 JS（ES Modules）· `mqtt`（公共 broker）· 自绘 Canvas 蜡烛图。
与 hearthStoneKill2 同栈，网络层 `MqttBus` 直接复用。

## 实现进度

- [x] **M1 单机 MVP**：行情回放 + 买卖 + 手续费 + 结算 + 排行
- [x] **M2 经济系统**：做多做空 + 保证金/爆仓 + 贷款（放款即计 + 每 30s 复利 + 欠债禁贷 + 结算扣本息）
- [x] **M3 多市场**：大A/港股/美股/虚拟货币多选 + 随机开局 + 盲盒
- [x] **M4 联机**：MQTT 房主权威（feed 逐 tick 推送）+ 房间管理（踢人/换座/增减 AI/调难度）
- [x] **M5 道具模式**：10 种道具 + 道具栏 + PvP + AI 会用道具
- [x] **M6 打磨**：真实历史数据接入（A股/港股/美股 = Yahoo Finance、加密 = Binance，合成走势兜底）+ 音效（Web Audio 合成，可静音并记忆偏好）+ 平衡性调参（全仓快捷键精确化、经典/自定义模式、移动端适配）

> 行情默认使用打包的**真实历史日 K**，随机抽标的与时间窗后做日内插值生成逐 tick 走势；数据缺失时自动回退到按市场波动率标定的合成走势（具备趋势/波动聚集/跳空）。数据见 `public/data/`，可用 `node scripts/fetch-data.mjs` 重新抓取。

## 代码结构

```
src/engine/   行情(market) 组合(portfolio) 贷款 做空 AI 道具 结算 —— 纯逻辑，无 DOM
src/net/      mqtt(MqttBus) online(房主权威联机)
src/ui/       lobby 房间 chart 交易面板 排行 道具 结算
```

> 历史行情仅作娱乐用途，费率/利率为可调的游戏平衡数值，不构成任何投资建议。
